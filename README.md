# 🔍 تَقَصّي AI | Taqassi AI
### بيئة ذكية لتحليل الأكواد واكتشاف الأخطاء | Intelligent Code Analysis & Debugging Environment

---

## 🌍 نظرة عامة | Overview
**تَقَصّي AI** هو تطبيق ويب متطور صممه المهندس حسن (خريج هندسة برمجيات - جامعة جدة) لمساعدة المطورين على فحص الأكواد، اكتشاف الثغرات المنطقية، وتقديم حلول فورية باستخدام الذكاء الاصطناعي.

**Taqassi AI** is an advanced web application developed by Engineer Hassan (Software Engineering Graduate - University of Jeddah) to help developers analyze code, detect logical flaws, and provide instant AI-powered solutions.

## ✨ المميزات | Features

* **التحليل الذكي (Smart Analysis):** اكتشاف الأخطاء وتقديم تقارير برمجية مفصلة.
    * *Detects errors and provides detailed programming reports.*
* **دعم اللغات (Multi-language):** يدعم React, Flutter, Python, Java, والمزيد.
    * *Supports React, Flutter, Python, Java, and many more.*
* **واجهة المستخدم (Pro UI):** يدعم الوضع الليلي، وضع المحيط، والوضع الفاتح.
    * *Features Dark, Ocean, and Light themes.*
* **تصدير التقارير (PDF Export):** إمكانية تحميل التحليل كملف PDF احترافي.
    * *Export analysis reports as professional PDF files.*
* **سجل العمليات (History):** حفظ آخر 10 عمليات تقصّي محلياً.
    * *Saves the last 10 analysis sessions locally.*

---

## 🛠 التقنيات المستخدمة | Tech Stack

* **Frontend:** React 19 (Vite).
* **Styling:** Tailwind CSS.
* **AI Engine:** Groq Cloud API (Llama 3.3 70B).
* **Syntax Highlighting:** React Syntax Highlighter.
* **Deployment:** Vercel (CI/CD).

---

## 💻 كيف يعمل؟ | How it Works?

1.  **Paste:** الصق الكود البرمجي في المحرر | Paste your code into the editor.
2.  **Select:** اختر لغة البرمجة المناسبة | Select the target programming language.
3.  **Analyze:** اضغط "بدء التقصّي" للحصول على الحل | Press "Start Analysis" to get the solution.
4.  **Download:** حمل التقرير كملف PDF | Download your report as a PDF file.
